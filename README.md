# time-sync-app

Простая утилита на Go для получения точного текущего времени с использованием NTP (Network Time Protocol).

## Описание

Программа обращается к публичному NTP-серверу (`pool.ntp.org`) и выводит точное текущее время в формате RFC3339.  
В случае ошибки соединения или обработки запроса - ошибка выводится в `stderr`, и программа завершается с ненулевым кодом выхода.

## Зависимости

Проект использует стороннюю библиотеку:
- [`github.com/beevik/ntp`](https://github.com/beevik/ntp) - для работы с NTP-протоколом.

## Обработка ошибок

Если не удаётся получить время от NTP-сервера (например, из-за отсутствия интернета или недоступности сервера), программа завершится с сообщением вида:

```text
2025/11/29 14:30:45 Ошибка при запросе времени NTP: ...
exit status 1
```
